CMFFormController

    CMFFormController replaces the portal_form form validation mechanism from
    Plone. It should work just fine in plain CMF as well.

Requires

    CMF 1.6+
    Zope 2.8.9+

Quickstart

    For CMF 1.6:
    1) Create an external method, module CMFFormController.Install,
       function: install
    2) Run it
